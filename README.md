  # Firmitas
  **Structural Analysis Suite for Modern Engineering**

  ## Overview
  Firmitas (Latin for Strength/Stability) is a high-performance toolkit built for civil and mechanical engineers. It streamlines the bridge between conceptual design and rigorous structural validation, providing real-time feedback on load distributions and material stress.

  ## Features
  * **Beam Calculator** - Compute reactions, shear force diagrams (SFD), bending moment diagrams (BMD), deflection, and peak stress.
  * **Truss Calculator** - Solve complex trusses using the method of joints or sections with automated tension and compression visualization.
  * **Export Ready** - Generate high-resolution PNG diagrams or export raw nodal data to CSV for external reporting.
  * **Data Management** - Save, version, and load project configurations directly from local storage.
  * **Adaptive UI** - A refined interface featuring full dark mode support for focused, long-duration engineering sessions.

  ## Tech Stack
  * Framework: React / Next.js
  * Styling: Tailwind CSS
  * Math Engine: Math.js / Custom Matrix Solvers
  * Visualization: Canvas API / Chart.js

  ## Quick Start
  Get your local development environment running in seconds.

  ### Prerequisites
  * Node.js (v18 or higher)
  * npm or yarn

  ### Installation
  ```bash
  git clone https://github.com/yourusername/firmitas.git
  cd firmitas
  npm install
  npm run dev
````
